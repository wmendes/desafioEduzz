
# desafioEduzz

## Passos para rodar o server:

 1. composer install
 2. copiar o arquivo .env.example para .env
 3. configurar seu banco de dados no arquivo .env
 4. php artisan migrate
 5. php artisan db:seed
 6. php artisan serve


## Passos para rodar o client:

 1. yarn install
 2. yarn start
